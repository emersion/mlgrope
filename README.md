# mlgrope

Cut the Rope - MLG version

![Screenshot](https://imgur.com/OBCF4Qg.png)

## Building

Requires Linux, OCaml 4.04+ and dune.

```shell
make
```

## Usage

```shell
./mlgrope # Runs the menu
./mlgrope -play <level path> # Plays a level
./mlgrope -edit <level path> # Edits a level
```
